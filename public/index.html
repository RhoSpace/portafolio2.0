PROYECTOS

Ver HuellasApp

HuellasApp es una aplicación móvil centrada en generar una comunidad de usuarios que compartan contenido sobre animales extraviados para facilitar una busqueda de mascotas perdidas por otros usuarios.

Proyecto en conjunto con David Rivas, se utilizo el lenguaje Java, Firebase y una base de datos NoSQL para su construcción.

Ver Tintalizú

Página Web para presentar productos del emprendimiento artístico tintalizú creado por Liced Vargas.

En la construcción se uso el preprocesador de css Sass junto al lenguaje Javascript. Tiene un enfoque responsive para ser accesible desde móviles.

Ver Tablero Kiyo

Proyecto personal sobre el juego de tablero Pokémon, consta de una aplicación web que facilita el manejo de los datos del juego y el nuevo reglamento para el mismo.

Sitío construido con React, uno de los componentes más destacados se enfoca en la obtención de datos desde una API y otro en la creación de una ficha para cada jugador.
